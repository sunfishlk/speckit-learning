# 颜值测试 Web 应用原型

基于 spec.md 规格说明的响应式颜值测试 Web 应用原型，支持移动端和桌面端。

## 🚀 快速开始

### 本地预览
1. 直接用浏览器打开 `index.html` 文件即可预览
2. 或者使用本地服务器（推荐）：
   ```bash
   # 使用 Python (Python 3)
   python -m http.server 8000

   # 使用 Node.js (需安装 http-server)
   npx http-server

   # 使用 Live Server (VS Code 扩展)
   ```

## 🎨 设计规范

### 响应式断点
- **移动端**: 320px+ (基础体验)
- **平板**: 768px+ (增强布局)
- **桌面**: 1024px+ (完整体验)
- **大屏**: 1440px+ (优化展示)

### 性能目标
- ✅ 页面加载时间 <3秒 (3G 网络)
- ✅ UI 交互响应 <100ms
- ✅ 触摸目标 ≥44px
- ✅ WCAG 2.1 AA 无障碍支持

## 🛠️ 技术栈

- **HTML5**: 语义化结构
- **CSS3**:
  - Flexbox & Grid 布局
  - 移动优先响应式设计
  - CSS 动画和过渡
  - 深色模式支持
- **JavaScript (ES6+)**:
  - 模块化类架构
  - 事件处理
  - 文件处理 API
  - 模拟 AI 分析

## ✨ 核心功能演示

1. **上传照片**: 支持点击选择或拖拽上传
2. **智能验证**: 自动检测文件类型、大小和图片质量
3. **模拟分析**: 仿真 AI 分析过程，包含进度动画
4. **结果展示**:
   - 颜值评分 (1-100)
   - 评分等级和详细说明
   - 五官特征分析
5. **明星对比**: 根据分数范围显示对应明星
6. **改善建议**: 分类个性化建议
7. **社交分享**: 支持 Web Share API
